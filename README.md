# Responsive Portfolio Website

A responsive personal portfolio website built using HTML, CSS, Bootstrap, and JavaScript.

## Features

- Responsive Design
- Smooth Scrolling Navbar
- Home Section
- Skills Section
- Projects Section
- Contact Section
- Download Resume Button
- Mobile Friendly Layout

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
